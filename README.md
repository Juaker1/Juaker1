## 🌟 Proyectos Destacados

### 🧠 [DocuMind](https://github.com/Juaker1/DocuMind)
**Sistema RAG Local para consulta inteligente de documentos.**
- Permite interactuar con archivos PDF de manera privada mediante LLMs locales.
- **Stack:** Next.js, FastAPI, PostgreSQL (pgvector), Ollama.

### 🧬 [Bioinformatics Platform](https://github.com/Juaker1/Proyecto-Toxinas)
**Trabajo de Título: Descubrimiento de analgésicos no opioides.**
- Análisis de péptidos de venenos de araña, procesando y curando datos para reducir **1348 candidatos a 23**.
- **Stack:** Python, Flask, SQLite.

### 📋 [Full-Stack Task Manager](https://github.com/Juaker1/task-manager) 
**Gestión reactiva de tareas con Angular & Hono.js** 
- Implementación de **Angular Signals** para una gestión de estado reactiva y eficiente, junto con estrategias de detección de cambios `OnPush`. 
- **Stack:** Angular, Hono.js, Drizzle ORM, SQLite.
